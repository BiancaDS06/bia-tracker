# bia-tracker⏱️

Este é um simples cronômetro desenvolvido com Vue.js. Ele permite a contagem do tempo para tarefas.

## 🚀 Tecnologias Utilizadas

- Vue.js 3
- HTML5
- CSS3
- JavaScript
- TypeScript

## 🎯 Funcionalidades

- ✅ Iniciar contagem do tempo
- 🌙 Alternar entre modo claro e escuro (se houver)
